# Security Policy

## Supported Versions

Security fixes are currently accepted for the latest released minor version.

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing
laharitadepalli183@gmail.com.

Include:

- the affected version or commit,
- a minimal reproduction or trace fixture when possible,
- impact and expected behavior,
- whether the report may be publicly credited after a fix.

Do not open public GitHub issues for sensitive reports. Expect an initial
response within 7 days. Confirmed vulnerabilities will receive a fix plan,
release target, and public advisory once disclosure is safe.
